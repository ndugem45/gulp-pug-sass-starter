
# Gulp-Pug-Sass Starter Project
==============================================================

A starter project for frontend templating


Project Structure :

```
|-- site/  (html files)
|-- src/
|
|------ assets/  (all assets files)
|------ pug  (page files)
|--------|
|--------|--- parts  (componets)
|
|------ sass  (sass files)
|--------|
|--------|--- parts  (sass component files)
|--------|--- pages  (sass page files)
```


## Installation

Install with npm

```bash
  npm i @ndugem45/gulp-pug-sass-starter
```
    
## Run Locally

Install dependencies

```bash
  npm install
```

Start the server

```bash
  npm start
```

Build project

```bash
  gulp build
```
