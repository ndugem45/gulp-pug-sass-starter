Gulp-Pug-Sass Starter Project
==============================================================

A starter project for frontend templating

==============================================================

Instalation :

```
npm i @ndugem45/gulp-pug-sass-starter
```

Project Structure :

```
__site  (html files)
__src
    __assets  (all assets files)
    __pug  (page files)
        parts  (componets)
    __sass  (sass files)
        parts  (sass component files)
        pages  (sass page files)
```

Command :

Run project
```
npm start
```

Build project
```
gulp build
```
